# Backpack 空投计算器

一款为 Backpack 生态用户打造的极简估值工具：用“积分”推演“空投价值”。输入你的积分与全网积分，几秒钟搞定个人空投价值与占比预估。**默认假设：FDV = 5B、空投比例 = 10%**（可随时调整）。

---

## 🧭 使用方式

1. **输入数据**
   - **我的积分**：你在 Backpack 生态中的当前积分。
   - **全网积分总量**：从公开数据/社区统计处获取的全网累计积分。
   - **可调参数（可选）**：
     - **FDV（Fully Diluted Valuation）**：默认 **$5,000,000,000**。
     - **空投比例**：默认 **10%**。

2. **查看结果**
   - 计算器会展示：
     - **1 分的预测价值（$ / 分）**
     - **我的空投价值（$）**
     - **我的积分占比（%）**

3. **计算逻辑（透明可检验）**
   ```text
   空投池总价值($) = FDV × 空投比例
   1 分预测价值($/分) = 空投池总价值 ÷ 全网积分总量
   我的空投价值($) = 1 分预测价值 × 我的积分
   我的占比(%) = 我的积分 ÷ 全网积分总量 × 100%
   ```

> ⚠️ 提示：本工具为**估算模型**，仅供研究和策略推演；实际发放、TGE 流动性与市场波动都会影响结果。
